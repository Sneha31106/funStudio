## 🏛️ Project Vision

**funStudio** (originally *Public API Playground*) is a high-end web application that transforms standard public API data into a refined, modernist interface. It prioritizes visual excellence through glassmorphic elements, classic typography, and subtle micro-animations, proving that utility and luxury can coexist.

---

## 💎 Key Features

- **Luxury-Minimalist Aesthetic:** A clean, sophisticated design inspired by modern architectural restraint.
- **Dynamic Feedback:** Real-time cursor-tracking glows and floating metallic sparkles in silver, pink, and green.
- **Glassmorphic UI:** High-fidelity cards featuring blurred backgrounds, modern shine effects, and smooth scale transitions.
- **Refined Typography:** A classic pairing of **Cinzel** (metallic display), **Playfair Display** (traditional serif), and **Inter** (clean functional sans).
- **Smooth Orchestration:** Seamless fade-up entry animations and elegant loading states for all data interactions.

---

## 🔗 Integrated Data Nexus

The application curates data from four distinct public sources:

1.  **🐕 Dog Finder:** Discover canine companionship via the *Dog CEO API*.
2.  **🎭 Joke Generator:** A subtle curation of algorithmic humor via the *Official Joke API*.
3.  **👤 Random User Profile:** Glimpse into synthetic identities generated by the *Random User API*.
4.  **📊 Data Explorer:** Query structured lorem ipsum artifacts from the *JSONPlaceholder API*.

---

## 🛠️ Tech Stack & Architecture

- **Core:** HTML5, CSS3 (Vanilla), JavaScript (ES6+ Modules)
- **Build Tool:** [Vite](https://vitejs.dev/) for ultra-fast development and optimized production builds.
- **Design:** Modern CSS techniques including `backdrop-filter`, `mask-image`, and sophisticated `cubic-bezier` transitions.
- **State Management:** Reactive DOM updates orchestrated with standard JS Fetch API.

## 🚀 Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18+ recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Sneha31106/funStudio.git
   cd funStudio
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
